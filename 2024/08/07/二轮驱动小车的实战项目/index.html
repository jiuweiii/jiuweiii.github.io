嵌入式系统常见基本概念

各种U

  • CPU

    CPU(Central Processing Unit)
    系统的大脑，功能是执行指令和处理数据

  • MPU

    Micro Processor Unit
    CPU的升级版

  • MCU

    微控制单元(Microcontroller Unit；MCU) ，又称单片机
    集成了CPU、存储器、定时器、各种寄存器和IO接口及其外设，能够实现一定的功能，但是是芯片级的系统

  • SOC

    单片系统或片上系统（英语：System on a Chip)
    是系统级别的芯片，可以运行操作系统

  • GPU

    graphics processing unit

    图形处理器，专门处理图像与图形运算的器件，给CPU分担工作的，减轻显卡对CPU的依赖

  • 上述的区别和联系
    CPU是计算机系统的核心组件，MPU是CPU的升级版，MCU是一块芯片中集成了整个计算机系统，就是核心组件加上了存储器，和一些外设，再加上简单的外围器件（电阻电容）就能执行代码了

  • DSP
    DSP 即数字信号处理技术， DSP 芯片即指能够实现数字信号处理技术的芯片，专注于数字信号的运算

  • PLC
    特点是稳定，MCUI单片机超负荷会烧掉，PC服务器没法胜任多路IO的输出，所以PLC一般用于设备的底层控制。

编码器
  • 定义
    把一定规律的信息或者数据转换为可以用于通讯、传输、储存的信号形式，称之为编码；反之，信号变成一定规律的信息或者数据是译码。
  • 增量式霍尔编码器
    比如说霍尔码盘转一圈，产生13个脉冲，霍尔码盘转30圈，机电主轴就旋转一圈，从而可以通过脉冲数来计算速度，来记录角位移
    • 四倍频
      霍尔编码器输出有A相和B相两条线，如果只监测A相的上升沿，就是上面说的那种；
      如果同时监测AB相的上升沿和下降沿，就是所谓的四倍频，会产生四倍的脉冲数，可以提高监测的精度。
    • 理解
      就相当于是，每个上升沿和下降沿视为一个刻度，当运动到这个刻度的时候，就触发一个脉冲，从而来计算位移。
    • 用途
      多用于计算监测电动机的转速的旋转方向
储存器

  • ROM，现在都是EEPROM，名字是电可擦可写入只读存储器，ROM是只读存储器，前面的修饰字母只是说明擦除的方式不一样。

  • flash属于EEPROM

  • 相对比如可不可读，我觉得，用RAM是断电了就会清空数据，ROM断电了，其数据也能永久保存来记忆更合适

黑白灰盒测试

  • 黑盒测试
    定于上说是把程序当成一个黑盒子，不知道内部的程序，只通过程序的表象来进行测试。实质上就是直接上手用这个程序，看看会不会出bug。

  • 白盒测试
    定义上说是把程序当成一个透明的盒子，只能看见内部代码，看不见外部的表面，实质上就是直接看代码有条理地debug。

  • 灰盒测试

    定义上是介于黑白的，实质上是产品快要完成前的最终测试，什么都要测一下。

电机部分

电机驱动
  1. 原理
    使用定时器输出PWM波来驱动电机
  2. 为什么要额外用一个电机驱动芯片
    stm32板子的电压是3.3V，但是电机是要12V，所以要把电信号放大，所以还要再额外加一个电机驱动芯片
